# react

与vue区别（为什么要学react）

7、框架本质不同

Vue本质是MVVM框架，由MVC发展而来；

React是前端组件化框架，由后端组件化发展而来。

React更偏向于构建稳定大型的应用，非常的科班化。

相比之下，Vue更偏向于简单迅速的解决问题，更灵活，不那么严格遵循条条框框。

因此也会给人一种大型项目用React，小型项目用Vue的感觉。
