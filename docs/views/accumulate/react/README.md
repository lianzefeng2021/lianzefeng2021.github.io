# react 

> 与vue区别（为什么要学 react ）

> 7、框架本质不同

Vue 本质是 MVVM 框架，由 MVC 发展而来；

React 是 前端组件化框架，由 后端组件化 发展而来。


> React 更偏向于构建稳定大型的应用，非常的科班化。

> 相比之下，Vue 更偏向于简单迅速的解决问题，更灵活，不那么严格遵循条条框框。

> 因此也会给人一种大型项目用 React，小型项目用 Vue 的感觉。
