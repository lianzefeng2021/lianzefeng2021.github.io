---

> String.prototype.valueOf()

> valueOf() 方法返回 String 对象的原始值

``` js
const stringObj = new String('foo');

console.log(stringObj);
// expected output: String { "foo" }

console.log(stringObj.valueOf());
// expected output: "foo"
```

> __描述__
>
>> 此方法通常由 JavaScript 在内部调用，而不是在代码中显式调用。
