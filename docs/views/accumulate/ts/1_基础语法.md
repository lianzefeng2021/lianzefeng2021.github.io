## 01. 基础语法

> TypeScript 面向对象编程实例：

```js
class Site {
  name (): void {
    console.log('Runoob')
  }
}
var obj = new Site()
obj.name()
```

> 以上实例定义了一个类 Site，该类有一个方法 name()，该方法在终端上输出字符串 Runoob。

> new 关键字创建类的对象，该对象调用方法 name()。

> 编译后生成的 JavaScript 代码如下：

```js
var Site = /** @class */ (function () {
  function Site () {}
  Site.prototype.name = function () {
    console.log('Runoob')
  }
  return Site
})()
var obj = new Site()
obj.name()
```
